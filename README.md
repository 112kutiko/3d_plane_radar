## Description

3d plane radar show plane position in 3d map 

## devices you need

1. rtl sdr and atenna.
2. latop or pc.

## required programs

1. dump1090
2. virtual radar
3. unity (tempory)

## How To Use

- on dump1090.
- on virtual radar.
- open virtual radar add dump1090 source.
- open unity project play and paste virtual radar link with AircraftList.json file.
- enter.
